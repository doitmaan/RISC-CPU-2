# RISC-CPU-2
A RISC CPU with modified bits width and instructions 
RS, RT, and RD are now 3 bits (to access 8 registers)
• Function is also 3 bits
• CPU data-path is now 6 bits (output from REGFILE)
• RAM address-path is now 6 bits
• 64 rows x 6 bits
• ROM address-path is now 12 bits, INSTRUCTION is 16 bits
• 4096 rows x 16 bits
 
 Th CAD program LOGISIM can be found in the the RISC CPU REPOSITORY
